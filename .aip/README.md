# .aip

本目录由 AIP 初始化生成，存放该仓库 AIP 机制的全部产出（隐藏工具目录，类比 .git）。

- `STATUS.md`：现状真理源（实现到哪/缺口/部署）——新会话先读这页
- `canonical-assets.md`：正典构件登记（造新前先查，防堆积）
- `decisions.md`：架构决策记录（ADR-lite，append-only）
- `findings.md`：侧发现收件箱（开发时撞见的无关问题，捕获别追）
- `knowledge.md`：知识库（验证过的根因/坎/领域事实，append-only，先查后挖）
- `knowledge_index.md`：知识索引（自动生成，勿手改，`aip knowledge` 重建）
- `config.yaml`：本项目适配配置（真理源/机器闸门命令/适用 lens）
- `_runtime/`：当前任务指针
- `features/`：功能工作包
- `protocols/`：项目内协议副本
