# .aip

本目录由 AIP 初始化生成，存放该仓库 AIP 机制的全部产出（隐藏工具目录，类比 .git）。

活文档（跨任务线、长期存在）：

- `OVERVIEW.md`：多线看板 + 自动摘要——开始/接手任何任务前先读这页
- `decisions.md`：架构/方向级决策记录（轻量、只追加）
- `knowledge.md`：知识库（验证过的根因/坎/领域事实，只追加，先查后挖）
- `knowledge_index.md`：知识索引（自动生成，勿手改，`aip knowledge` 重建）
- `reference.md`：本工程权威参照（领域概念/术语、核心铁律、可复用实现）
- `inbox.md`：旁路问题收件箱（干 A 时撞见的无关问题，捕获别追）
- `conventions.md`：项目规约（怎么干的常驻规则）
- `config.yaml`：本项目适配配置（真理源/机器检查命令/适用视角）

任务线状态记在 `OVERVIEW.md` 看板，不再有 per-feature 工作包目录或运行时指针。
